Create `evidence/evidence_projection.py` with the following content:

1. Module docstring: `"""evidence_projection.py — Agent-facing evidence projection models. Debate agents consume EvidenceProjection instead of raw EvidenceBundle. This is the adapter boundary between the evidence/retrieval layer and the debate layer."""`

2. Imports: `from typing import Literal, Optional` and `from pydantic import BaseModel, Field`

3. Define `EvidenceSummaryCard(BaseModel)` with these exact fields:
   - `card_id: str = Field(description="卡片标识，格式: 'card_{rule_id}'")`
   - `question: str = Field(description="该卡片回答的核查问题（自然语言）")`
   - `finding: str = Field(description="取证发现摘要（自然语言）")`
   - `status: Literal["supports", "contradicts", "unresolved", "missing"] = Field(description="证据状态：supports=支持通过, contradicts=反对通过, unresolved=需要判断, missing=证据缺失")`
   - `confidence: float = Field(ge=0.0, le=1.0, description="该证据项的置信度")`
   - `artifact_refs: list[str] = Field(default_factory=list, description="溯源引用列表（evidence_id 或 rule_id）")`

4. Define `EvidenceProjection(BaseModel)` with these exact fields:
   - `task_header: str = Field(description="任务标题，如'灵活就业社保补贴资格认定'")`
   - `target_person: str = Field(description="被核查人员标识（身份证号）")`
   - `policy_scope: str = Field(description="政策范围，如'灵活就业社保补贴'")`
   - `cards: list[EvidenceSummaryCard] = Field(default_factory=list, description="证据摘要卡片列表")`
   - `uncertainty_markers: list[str] = Field(default_factory=list, description="不确定性标记列表")`
   - `total_cards: int = Field(default=0, description="证据卡片总数")`
   - `resolved_count: int = Field(default=0, description="已确认状态(supports/contradicts)的卡片数")`
   - `unresolved_count: int = Field(default=0, description="未确认状态(unresolved/missing)的卡片数")`

No other logic or functions in this file — pure data models only.
