# Plan 03: Qualification-Aware Question Template Registry

## Goal

实现 `cognition/question_templates.py`，建立可复用的核查问题模板库。模板在组织方式上按问题类型归类，但语义上仍然保持 qualification-item-first，覆盖 BASIC / EXCL / INFER / CALC 四类，并为后续 `evidence_planner` 提供稳定输入。

## Verification

- `cognition/question_templates.py` 存在且可导入
- 模板至少覆盖 BASIC / EXCL / INFER / CALC
- 每个模板都能追溯到 qualification item / question id / expected answer shape
- `python -m pytest tests/test_question_templates.py -q` 通过

## must_haves

- 组织上按问题类型，语义上不丢掉 qualification item
- 模板不直接绑定 SQL，不退化为静态规则表副本
- 模板中有 `question_text`、`question_type`、`suggested_evidence_targets`、`time_semantics_hint`
- 支持后续 planner 通过 question id 或 qualification item 进行检索
- 与现有前端和 debate 主链完全隔离
